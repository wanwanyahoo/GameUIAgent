# 定价、积分与订阅规则

## 1. 目标

平台需要复刻 VberAI 的 AI Studio 订阅、积分、API 权限和项目额度体系，同时为后续商业化预留 Stripe 支付、自动续费、升级和降级规则。

## 2. 套餐

### 2.1 Free

- 免费。
- 每日免费积分。
- 月度积分。
- 1 个并发 AI 任务。
- 支持 AI Studio 基础能力。
- 本地项目额度。

### 2.2 Base

- 月付/年付。
- 更多每日积分。
- 更多月度积分。
- 3 个并发 AI 任务。
- 1 个云项目。
- 2 个本地项目。
- Base AI 模型和 AI 功能。
- 支持将设计稿直接导出为引擎场景和 Prefab。

### 2.3 Plus

- 5 个并发 AI 任务。
- 更多云项目和本地项目。
- 全部 AI 模型和 AI 功能。
- 支持设计稿直接导出引擎场景和 Prefab。
- 15 分钟企业级加密热备份快照。

### 2.4 PRO

- API 调用 AI 服务。
- 更高请求速率。
- 8 个并发 AI 任务。
- 更多云项目和本地项目。
- 全部 AI 模型和 AI 功能。
- 支持设计稿导出引擎场景和 Prefab。
- 支持从游戏项目直接导入场景和 Prefab 为可编辑画布。
- 5 分钟企业级加密热备份快照。

### 2.5 MAX Expert Program

- 20 个并发 AI 任务。
- 大量云项目和本地项目。
- 全部 AI 模型和 AI 功能。
- 支持设计稿导出引擎场景和 Prefab。
- 支持从游戏项目导入场景和 Prefab。
- 工作日专家客服优先支持。
- 3 分钟企业级加密热备份快照。

## 3. 积分类型

### 3.1 Daily Free Credits

- 随订阅计划赠送。
- 每天 00:00 重置。
- 未使用部分不结转。
- 消耗时优先使用。

### 3.2 Monthly Credits

- 随订阅计划赠送。
- 每月 1 日重置。
- 年付订阅可获得额外月度积分。
- 未使用部分不结转。

### 3.3 Purchased Credits

- 用户单独购买。
- 永不过期。
- 多次购买可累加。
- 在每日和月度积分耗尽后使用。

## 4. 扣费顺序

系统自动按以下顺序扣除：

1. Daily Free Credits。
2. Monthly Credits。
3. Purchased Credits。

## 5. 订阅规则

### 5.1 自动续费

- 首次付费订阅后默认开启自动续费。
- 到期前自动扣费。
- 用户可在账号设置关闭自动续费。
- 取消后服务持续到当前周期结束。

### 5.2 升级

- 用户可随时升级到更高套餐。
- 月付用户可升级到更高月付或任意年付。
- 年付用户只能升级到更高年付。
- 升级立即生效，费用按当前剩余价值折算。

### 5.3 降级

- 不支持直接降级到低级套餐。
- 年付用户不能直接降级为月付。
- 如需降级，需要先取消自动续费，当前周期结束后重新选择。

### 5.4 月付转年付

- 月付用户可随时切换为同档年付。
- 切换立即生效并按比例折算。
- 年付通常提供更大折扣和更多积分。
- 年付用户不能直接切回月付。

## 6. 支付集成

- 使用 Stripe 或等价支付网关。
- 支持订阅 Checkout。
- 支持支付成功回调。
- 支持订阅状态同步。
- 支持发票和账单记录。

## 7. 用量限制

套餐需要控制：

- 每分钟请求数。
- 每日请求数。
- 并发 AI 任务数。
- 云项目数量。
- 本地项目数量。
- API 调用权限。
- 热备份快照频率。

## 8. API 限流

开发者 API 响应需要包含：

- `X-RateLimit-Limit`
- `X-RateLimit-Remaining`
- `X-RateLimit-Reset`

限流命中时返回 `429 Too Many Requests`。

## 9. 错误码

- `INSUFFICIENT_CREDITS`
- `PLAN_NOT_ALLOWED`
- `RATE_LIMIT_EXCEEDED`
- `SUBSCRIPTION_EXPIRED`
- `PAYMENT_REQUIRED`
- `AUTO_RENEW_DISABLED`
- `DOWNGRADE_NOT_ALLOWED`

## 10. 验收标准

- 用户能查看套餐。
- 用户能查看每日、月度、购买积分余额。
- AI 任务能按优先级扣积分。
- Developer API 能返回限流 Header。
- 订阅升级、取消自动续费有明确状态。
- PRO/MAX 支持从游戏项目导入场景和 Prefab 的套餐权益标记。
